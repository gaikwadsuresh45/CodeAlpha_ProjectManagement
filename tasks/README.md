# 📋 TaskFlow - Project Management Tool

## CodeAlpha Full Stack Development Internship — Task 3

### 🔗 Project Overview
A Trello-like Project Management web application built with Django.

### ✅ Features
- Project boards with task cards
- To Do → In Progress → Done status
- Task assignment to team members
- Comments system on tasks
- User Login & Register
- Admin dashboard

### 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Django (Python)
- **Database:** SQLite

### 🚀 How to Run
```bash
pip install django
python manage.py migrate
python manage.py runserver
